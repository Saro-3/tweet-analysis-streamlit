# Tweet Analysis & Visualization Web App

A Streamlit-based web application for searching, analyzing, and visualizing tweet data.

## 🚀 Features
- Search tweets
- Data filtering
- Interactive charts using Plotly
- CSV-based dataset processing
- Simple and responsive UI

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly

## 📂 Project Structure
- app.py – main application
- search.py – search logic
- page_config.py – UI configuration
- Tweet.csv / Tweets.csv – datasets

## ▶ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
